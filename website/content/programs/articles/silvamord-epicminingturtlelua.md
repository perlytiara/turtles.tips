**What it does**  
EpicMiningTurtle is a program for CC:Tweaked turtles designed for advanced mining tasks in Minecraft.

**Setup**  
- Equip a mining turtle with fuel (e.g., coal) and any necessary tools.  
- Download the program in-game using `wget https://raw.githubusercontent.com/Silvamord/EpicMiningTurtle.lua` (assuming GitHub repo access).

**Usage**  
Run the program with `EpicMiningTurtle` (or the filename if renamed). Follow on-screen prompts for configuration options like mining area size or depth. For full details, watch the embedded video.

**Tips**  
Ensure the turtle has enough inventory space for ores and starts in a safe position to avoid getting stuck.
