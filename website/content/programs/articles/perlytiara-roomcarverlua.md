**What it does**  
This program enables a CC:Tweaked turtle to carve out rooms in your world. For full details, watch the embedded video.

**Setup**  
- Equip a mining turtle with sufficient fuel (e.g., coal).  
- Download the program in-game using `wget` or by copying from the path: perlytiara/room_carver.lua (e.g., from a GitHub repository).

**Usage**  
Run the program with `room_carver` and follow any on-screen prompts for room dimensions or options. Refer to the video for examples.
