**What it does**

Turtlogenesis is a program for CC:Tweaked turtles, created by MerlinLikeTheWizard. Watch the embedded YouTube video for a demonstration and full explanation of its functionality.

**Setup**

- Equip a turtle with fuel (e.g., coal or lava).
- Download the program in-game using:  
  `wget https://raw.githubusercontent.com/MerlinLikeTheWizard/turtlogenesis/main/turtlogenesis.lua`

**Usage**

Run the program by typing `turtlogenesis` in the turtle's terminal. Refer to the video for main options and how to use it effectively.
