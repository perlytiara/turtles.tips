**What it does**  
This program enables a turtle to perform automated quarry mining in CC:Tweaked. For full details, watch the embedded video.

**Setup**  
- Equip a mining turtle with fuel (e.g., coal) and any necessary tools.  
- Download the program in-game: `wget https://raw.githubusercontent.com/BigGamingGamers/quarry-miner.lua` (adjust URL if needed based on the repository).

**Usage**  
Place the turtle at the starting position, then run `quarry-miner` with optional arguments for size or depth (refer to the video for specifics). The turtle will mine and return resources.
