**What it does**  
This program by BigGamingGamers is designed for CC:Tweaked turtles; see the embedded video for details on its functionality related to updating lifts.

**Setup**  
You'll need a turtle with fuel. Obtain the program via its path: BigGamingGamers/update_lift.lua (e.g., using wget if it's a downloadable Lua file). Refer to the video for specific setup requirements like blocks or tools.

**Usage**  
Run the program with `lua update_lift.lua` or as instructed in the video. Main options and commands are demonstrated there.
