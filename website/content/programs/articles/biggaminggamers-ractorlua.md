### What it does
This program, `ractor.lua`, is designed for CC:Tweaked turtles, created by BigGamingGamers. For details on its functionality, watch the embedded YouTube video.

### Setup
- Ensure you have a turtle with fuel and any required peripherals or blocks as shown in the video.
- Download the program in-game using: `wget https://raw.githubusercontent.com/BigGamingGamers/ractor.lua` (adjust URL if needed, assuming GitHub repo).

### Usage
Run the program with `lua ractor.lua` or as directed in the video. Check the video for main options and commands.
