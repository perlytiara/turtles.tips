**What it does**

bMobFarm is a CC:Tweaked turtle program by Kaikaku for building a mob farm. See the embedded video for a full demonstration.

**Setup**

- Prepare a turtle with fuel and any required building materials (details in the video).
- Download the program via wget using the path Kaikaku/bMobFarm.lua (e.g., from GitHub: `wget https://raw.githubusercontent.com/Kaikaku/bMobFarm/main/bMobFarm.lua` or similar; confirm the exact URL in the video).

**Usage**

Run the program with `bMobFarm` and follow on-screen prompts or options shown in the video.

**Tips**

Watch the video for setup variations and optimization tips.
