### What it does
This turtle program allows turtles to automatically build three different mansion structures in Minecraft, showcasing advanced building automation in ComputerCraft.

### Setup
- You'll need a fueled turtle equipped with sufficient building blocks (e.g., stone, wood, glass) in its inventory, placed in a suitable open area.
- Download the program in-game using: `pastebin get PfdurUkb bMansion`.

### Usage
Run the program with `bMansion` to start building one of the three mansion designs. For details on selecting specific mansions or required materials, refer to the embedded video.
