**What it does**  
Turtle-manager is a CC:Tweaked program for managing turtles, created by perlytiara. For full details on its functionality, watch the embedded video.

**Setup**  
- Requires a turtle with fuel and access to the HTTP API (enabled in CC:Tweaked config).  
- Download from GitHub at perlytiara/turtle-manager. In-game, use `wget` to fetch files (e.g., `wget https://raw.githubusercontent.com/perlytiara/turtle-manager/main/manager.lua` assuming standard repo structure).

**Usage**  
Run the program on your turtle (e.g., `manager`). Refer to the video for main options and commands.

**Tips**  
Ensure your turtle has sufficient fuel for operations; check the GitHub repo for updates or additional scripts.
