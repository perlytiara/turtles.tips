**What it does**

Minecraft-Toolkit is a collection of CC:Tweaked turtle utilities focused on movement, mining helpers, refueling, and quick in-game setup via `updater.lua`.

**Setup**

- Place the toolkit files in your turtle computer.
- Run `updater` to download/update the script set from the upstream repository.
- Keep fuel available before running movement-heavy tools.

**Usage**

Start with `tk` for the legacy toolkit entrypoint, or run individual scripts such as `cobble`, `flint`, and `lavaRefuel` directly depending on your task.
