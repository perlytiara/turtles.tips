**What it does**  
This program sets up a phone server in CC:Tweaked, enabling communication features for turtles or computers (details in the video).

**Setup**  
- Requires a turtle or computer with modem access.  
- Download via in-game command: `wget https://raw.githubusercontent.com/BigGamingGamers/phone-server.lua` (adjust URL if needed) or use pastebin if available.

**Usage**  
Run the program with `lua phone-server.lua`. Follow video for configuration options and integration with phone clients.

**Tips**  
Watch the embedded video for setup examples and troubleshooting.
