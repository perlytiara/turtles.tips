**What it does**  
@load is a ComputerCraft program by BigGamingGamers for turtles in CC:Tweaked. For full details, watch the embedded video.

**Setup**  
- Requires a turtle or computer in CC:Tweaked.  
- Download the program using wget in-game: `wget https://raw.githubusercontent.com/BigGamingGamers/some-repo/main/@load.lua` (adjust URL based on the author's repository for BigGamingGamers/@load.lua).

**Usage**  
Run the program with `lua @load.lua` or as instructed in the video. Check the video for main options and commands.
