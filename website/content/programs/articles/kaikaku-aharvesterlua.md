**What it does**  
aHarvester is a ComputerCraft turtle program by Kaikaku for automated harvesting tasks. Watch the embedded video for full details.

**Setup**  
- Prepare a fueled turtle with any required tools or inventory slots (refer to video).  
- Download via in-game command: `wget https://raw.githubusercontent.com/Kaikaku/aHarvester.lua` (adjust URL if needed) or use pastebin if available.

**Usage**  
Run the program with `aHarvester` and follow on-screen prompts or video instructions for options like area size or crop type.

**Tips**  
Ensure the turtle has fuel and clear space around the farm area for best results.
