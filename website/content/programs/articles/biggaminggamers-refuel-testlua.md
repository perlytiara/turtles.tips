**What it does**

refuel-test is a CC:Tweaked program by BigGamingGamers for testing turtle refueling. Watch the embedded video for full details.

**Setup**

You'll need a fueled turtle with the program installed. Download refuel-test.lua from the path BigGamingGamers/refuel-test.lua (e.g., via wget from a repository like GitHub).

**Usage**

Run the program in your turtle with `lua refuel-test.lua` or as configured. Refer to the video for options and execution.
