**What it does**  
This program allows a ComputerCraft turtle to create or manage a lift (elevator) system.

**Setup**  
Equip a turtle with fuel and necessary blocks. Download the program via `wget https://raw.githubusercontent.com/Michael-Reeves808/lift.lua lift` (adjust URL if in a specific repo).

**Usage**  
Run `lift` in the turtle's terminal. Refer to the video for configuration options and examples.

Watch the embedded video for setup details and troubleshooting.
