**What it does**

quarry.cc is a turtle program for CC:Tweaked, created by siphorns. Watch the embedded video for details on its functionality.

**Setup**

You'll need a turtle (e.g., mining turtle) in your CC:Tweaked world. Obtain the program from the path: siphorns/quarry.cc.lua (use in-game tools like wget if it's a downloadable URL, or copy it manually).

**Usage**

Run the program on your turtle and follow the options shown in the video for configuration and execution.
