**What it does**  
The ore-keeper program automates ore management for CC:Tweaked turtles. For full details on its functionality, watch the embedded video.

**Setup**  
- Equip a turtle with fuel (e.g., coal) and any required tools or blocks as shown in the video.  
- Download the program in-game using wget: `wget https://raw.githubusercontent.com/BigGamingGamers/ore-keeper.lua` (adjust URL if needed based on the author's repository).

**Usage**  
Run the program with `ore-keeper` followed by any options demonstrated in the video. Monitor the turtle's actions and adjust parameters as needed.

**Tips**  
Ensure the turtle has enough inventory space for ores. Refer to the video for troubleshooting common issues.
