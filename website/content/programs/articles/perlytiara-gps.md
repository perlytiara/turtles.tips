**What it does**  
This program by perlytiara implements GPS functionality for turtles in CC:Tweaked, allowing location tracking and navigation.

**Setup**  
- Equip a turtle with a wireless modem.  
- Ensure the turtle has fuel (e.g., coal).  
- Download the program in-game: `pastebin get perlytiara/gps gps` (or use wget if available).

**Usage**  
Run the program with `gps` followed by commands (e.g., `gps host` to set up a GPS host or `gps locate` for position). Refer to the embedded YouTube video for detailed options and examples.

**Tips**  
Set up multiple GPS hosts in a 3D grid for better accuracy. Watch the video for setup demonstrations.
