### Setup
- Ensure you have a turtle with sufficient fuel (e.g., coal or lava).
- Download the program using: `wget https://raw.githubusercontent.com/perlytiara/husbandry/main/husbandry.lua husbandry` (adjust the URL if the repository structure differs).

### Usage
Run the program with `husbandry` in the turtle's terminal. Refer to the embedded YouTube video for specific options, configuration, and how to operate it.

### Tips
Watch the video for setup examples and troubleshooting common issues like turtle positioning or inventory management.
