**What it does**  
This program enables a turtle to ignite TNT in Minecraft using CC:Tweaked (ComputerCraft). For details, watch the embedded video.

**Setup**  
- Equip a turtle with flint and steel (or redstone tools if needed) and sufficient fuel.  
- Download the program in-game: `wget https://raw.githubusercontent.com/BigGamingGamers/tnt-igniter/main/tnt-igniter.lua` (adjust URL if necessary based on the author's GitHub repo).

**Usage**  
Run the program with `tnt-igniter` and follow any on-screen prompts or video demonstration for placement and ignition options.
