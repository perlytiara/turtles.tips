### What it does
This program, `detectids.lua` by BigGamingGamers, allows a turtle to detect and handle item or block IDs in CC:Tweaked. For full details on its functionality, watch the embedded video.

### Setup
- Equip a turtle with fuel (e.g., coal) and any necessary tools or blocks as shown in the video.
- Download the program in-game using Pastebin or wget: `pastebin get [code] detectids` (replace [code] with the actual Pastebin ID) or directly from the path `BigGamingGamers/detectids.lua`.

### Usage
Run the program with `detectids` in the turtle's terminal. Follow the video for specific commands, options, and examples.

### Tips
Ensure the turtle has sufficient fuel and inventory space. Test in a safe area to avoid unexpected behavior.
