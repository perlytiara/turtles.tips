**What it does**

A CC:Tweaked turtle program for automating harvesting tasks, as shown in the accompanying YouTube video.

**Setup**

- Prepare a fueled turtle (refuel as needed for the task).
- Download the program via wget:  
  `wget https://raw.githubusercontent.com/Michael-Reeves808/harvest/main/harvest.lua harvest`

**Usage**

Run the program with `harvest` followed by any options demonstrated in the video. Refer to the video for configuration and running details.

**Tips**

Ensure the turtle has inventory space for harvested items. Watch the video for setup examples and troubleshooting.
