**What it does**

Computercraft-GUI is an upgraded turtle toolkit focused on a menu-driven interface with mouse support on advanced turtles and keyboard-only controls on standard turtles.

**Setup**

- Copy or download `installer.lua` or `tk3.lua` onto a turtle.
- Run the installer to fetch required `lib` and `scenes` modules.
- Keep HTTP enabled in CC:Tweaked so remote downloads can complete.

**Usage**

Run `tk3` to open the GUI toolkit, then navigate with clicks (advanced turtle) or indexed keyboard shortcuts (standard turtle) to launch utilities.
