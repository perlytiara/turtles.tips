**What it does**

tClear is a CC:Tweaked turtle program by Kaikaku that clears specified areas or blocks.

**Setup**

- Equip a turtle with fuel (e.g., coal) and any necessary tools.
- Download the program in-game via wget or pastebin to save as Kaikaku/tClear.lua (check the author's repository or video for the exact URL or code).

**Usage**

Rename the file to tClear.lua if desired, then run `tClear` with optional arguments for area size or direction (refer to the video for specifics).

**Tips**

Ensure the turtle has inventory space for collected items. Watch the embedded video for a full demonstration and configuration options.
