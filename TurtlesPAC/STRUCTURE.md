# TurtlesPAC folder structure

This folder lives inside **turtles.tips**. It contains only the program archive (submodules + docs). Website and **data/** are at the repo root.

```
turtles.tips/
├── TurtlesPAC/           ← this folder (archive only)
│   ├── programs/        # submodule — CC-Tweaked-TurtsAndComputers
│   ├── community/       # submodules
│   │   ├── README.md
│   │   └── ...
│   ├── docs/
│   ├── resources.md
│   ├── README.md
│   └── STRUCTURE.md
├── website/             # at repo root
├── data/                # at repo root
└── README.md
```

No duplicate names: **TurtlesPAC** is the only folder that holds the archive.
